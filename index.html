平台概述

骞云科技是云原生管理平台的倡导者和引领者，致力于构建AI赋能的面向自服务和DevOps的统一云原生管理体系，帮助客户构建数字企业。现有自主研发的六大云原生管理产品、SmartCMP SaaS平台和多种场景解决方案。骞云率先将ITSM+ITOM+ITBM相融合，帮助企业实现AI驱动的高度自动化的统一管理和极致体验，全面深度赋能企业数字化转型。

骞云云原生运维管理平台，能够管理企业的公有云、专属云、私有云、边缘云。平台通过统一的云组件模型，抽象任意的云资源，形成超级自动化运维的基础能力。对云上的每个云产品，平台提供包括资源发现、监控、运维变更、安全访问、自愈、持续优化等完整的全生命周期的能力。

骞云云原生运维管理平台有着业界最广泛的云平台接入。目前平台已经支持了四十种云平台的接入，不仅有阿里云、腾讯云、AWS等公有云，也有vSphere、OpenStack虚拟化平台，还可以接入Kubernetes容器化平台和X86裸金属服务器，以及VMware NSX、Cisco ACI、XSKY、F5 Big IP等网络、存储以及安全平台。平台具有通过云组件的强大扩展能力，有着业界最丰富的多达300+种云产品支持。既包括vSphere、OpenStack等私有云的IaaS资源，也有阿里云、AWS、腾讯云、Azure等公有云、专有云的IaaS和PaaS服务，更有数十种网络、存储、容器、软件等丰富的产品支持。

平台包括六大产品模块，包括云自动化管理、云持续交付、云监控告警、云费用分析、云堡垒机、云ITSM。

1. 云自动化管理产品

云自动化管理产品是完整的、面向混合IT环境的管理平台。可统一管理公有云、私有云和混合云，提供跨平台的自动化和编排能力，以及自服务所需要的策略、配额和流程控制。

平台核心功能：

  1. 统一管理多云环境：支持公有云，私有云，容器云等，并可批量导入存量云资源、自动同步更新在公有云控制台中创建的新资源。

  2. 标准化的服务目录，规范资源使用：通过服务目录规范化流程，策略和配置，控制用户使用符合IT部门安全和合规要求的标准等。

  3. 混合编排，实现端到端自动化：提供多云混合编排能力，将软件、SDN、超融合等资源和传统服务整合，帮助客户打破资源孤岛，实现完整应用的端到端自动化交付。

  4. 支持云中立，避免云锁定：平台基于各种策略自动选择，提供了真正的云中立服务能力。

  5. 全面多层次的统一运维：统一运维混合云场景中的IaaS，PaaS和软件，统一权限，统一体验。

  6. 资源全生命周期的自服务：方便业务部门用户按需获取多云环境资源，实现自助扩容、监控、回收资源，并可管控权限、审批、配额、回收策略等。

2. 云持续交付产品

云持续交付产品是面向公有云、私有云和混合云的企业级持续交付产品，集DevOps实践、业界前沿的研发理念，提供可视化流水线，通过高度自动化的服务编排、自服务，让软件开发简单高效。在DevOps的持续发布过程，把人工触发应用创建、更新，变成Git/Jenkins的事件来触发，结合权限管控和详细的结构展示，能够满足用户复杂多样的业务使用场景。

平台核心功能：

  1. 端到端的应用自动化部署：通过蓝图编排实现从简单到复杂的应用资源端到端交付，将各种面向应用、面向环境的部署规范落地到自动化交付过程。

  2. 面向开发测试场景的自服务：开发测试用户可方便的按需获取多云环境资源，自助扩容、监控、回收资源，并受到权限、审批、配额等管控。

  3. 应用持续交付流水线：开发测试人员可方便的自定义应用发布、测试、交付流水线。基于面向应用、面向多云的环境端到端部署和自助变更能力，开发测试人员可专注应用部署、变更的标准化、自动化。

  4. 资源全生命周期管理：开发测试人员可方便的按需获取多云环境资源，自助扩容、监控、回收资源，并受到权限、审批、配额、回收策略等管控。

  5. 集成多种软件制品：支持Nexus，Jfrog Artifactory，VMware Harbor等多种制品库，并将制品自动化更新到运行云环境。

3. 云监控告警产品

云监控告警产品是资源与服务一体化的监控告警产品，面向多云服务、现代应用，以ITIL流程为基础，以AIOps为驱动，集监控、告警、自动化运维、故障自愈的一体化监控告警产品。平台支持从不同的监控工具和平台采集运维数据，基于整个Prometheus生态体系，实现对云资源快速监控，也可以添加自定义Exporter扩展任意云资源的监控，支持物理设备、公有云、私有云、容器、数据库中间件等应用的全栈资源监控；支持灵活定义告警指标、告警策略，平台从监控聚合数据中，检查符合告警指标、告警策略的数据，判定符合后即触发告警，将通知到指定用户，支持以邮件、短信等多种形式，确保第一时间进行问题处理；支持灵活的告警处理流程，是ITIL流程和故障自愈的有机集成，支持多种自动化的故障自愈，从而降低故障处理时长，提升企业 IT 服务的可用性。

平台核心功能：

  1. 支持云资源的统一监控：平台支持物理设备、公有云、私有云、容器、数据库中间件等应用的全栈资源监控。通过云网关实现跨云、跨网络的统一监控，打破监控孤岛。

  2. 可扩展到任意云监控：整合Prometheus生态体系，实现对云资源（云主机、RDS、Redis、Nginx等）快速监控，也支持添加自定义Exporter扩展任意云资源的监控。

  3. 资源性能与健康分析：通过主页总览不同云资源使用与健康状态，支持以蜂窝图的形式，展示所有虚拟机运行状态，根据颜色深浅，直观判断部署的健康程度，辨别潜在热点。

  4. 灵活的告警策略和处理流程：支持多种告警策略，故障告警、阈值告警或预测告警。告警触发后，支持自动生成工单，通知到指定用户，确保第一时间进行问题处理。支持告警级别的设置和告警自动升级。

  5. 基于策略分析实现故障自愈：基于ITIL流程，支持多种自动化的故障自愈（故障工单、云资源自动化变更、Jenkins任务、外部系统集成等），从而降低故障处理时长，提升企业 IT 服务的可用性。

4. 云费用分析产品

云费用分析产品是面向混合云场景的多云费用分析产品，统一采集或生成公有云、私有云和混合云的各种费用，支持按组织架构分解并分析成本，并通过多种维度分析发现资源配置问题，并提供一键优化。平台还支持完善成本治理来减少浪费，从而实现云费用可追踪，可分析，可优化。提供从云平台到应用系统的立体化监控，实时采集资源性能，及时发现浪费情况，并通过灵活可配置的优化策略一键优化配置，减少不必要的浪费，实现成本节约。

平台核心功能：

  1. 建设统一的云费用管理体系：系统支持多种云平台接入，支持创建多层级的组织架构体系，实现云成本数据的统一管理。系统提供基于ITIL标准的费用审批流程，帮助企业落地费用管理制度。

  2. 汇总企业任意云成本：平台支持多种方式对云成本进行汇总，实现企业任意IT支出的统一核算，公有云支持账单自动同步，私有云支持自定义计费规则，针对非云资源，如租赁费、电费等，支持通过自定义方式导入。

  3. 自动分析云成本：平台支持配置灵活的费用清洗规则、费用分摊规则，自动实现部门费用分摊。平台实时计算各部门费用成本，当达到预算配额时及时产生告警，提醒用户控制成本支出。

  4. 可视化云成本：平台内置丰富的仪表盘组件，开箱即用，实时展示费用使用情况。内置多种费用报表，能够按照多种维度进行筛选和分组，提供不同角度的查看和对比。平台提供详细的费用报告，可精确跟踪每一笔支出。

  5. 云成本优化策略：平台提供从云平台到应用系统的立体化监控，实时采集资源性能，及时发现浪费情况。提供灵活可配置的优化策略，定期清理闲置云资源、非工作时间对资源进行关闭等，及时减少不必要的浪费，实现成本节约。

5. 云堡垒机

云堡垒机可集中管控多云环境下的云资源访问，通过脚本库，软件组件标准化企业的运维能力；全程记录操作数据，还原运维场景，助力企业用户构建云上统一、安全、高效的运维通道。

平台核心功能：

  1. 统一管理云资源访问凭证：支持通过修改访问凭证实现云资源的批量改密，无需手动输入账户和密码，用户可以通过访问凭证一键登录到设备。

  2. 完善的访问权限控制：提供多种权限的默认角色，也支持自定义不同角色，实现基于策略驱动的访问权限和最小权限运维。

  3. 实现高效运维：支持多种运维管理协议（SSH、RDP、VNC、 SFTP等），支持批量运行脚本 ，巡检主流数据库，自动化安装中间件等，提高运维效率。

  4. 提供全面的运维操作审计：支持登录审计、操作审计（用户操作行为）、指令审计（关键字过滤、高危命令告警等）、文件传输审计（文件上传、下载）等，支持用户查看操作记录和历史会话信息，并能够查看或终止当前进行的会话。

6. 云ITSM产品

云ITSM产品是云时代的IT服务门户，提供业务所用到的核心IT能力，可更好的支撑和引领业务；对IT技术的管理包括云服务管理、ITIL服务管理等。

平台核心功能：

  1. 支持多种工单类型：提供事件工单、变更工单、问题工单、请求工单等服务类型，内置对应的表单参数和流程，满足企业IT服务管理的需求。

  2. 丰富的工单分配方式：支持多种处理人进行工单处理，包括服务团队、角色成员、个人或申请者；服务团队成员可进行认领或由系统进行自动工单分配；

  3. 完善的工单处理功能：处理工单时，支持创建关联工单，或关联已有的工单，厘清多个工单的关系，多个工单重复，或变更工单导致问题工单的产生等；支持升级、转派、退回等多种工单处理操作；支持处理工单添加备注说明情况，可将备注设为仅管理员可见。

  4. 独立的工单控制台: 提供ITSM控制台和各种类型工单的菜单，方便快速、直接地进行查看、处理或提单；

  5. 支持排班管理功能：管理员能够添加并管理服务团队的排班，启用排班表，待处理的工单可被分配给值班人员，提升处理工单的效率。
